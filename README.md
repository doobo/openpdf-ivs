# openpdf-ivs

> 
## 如何添加,注意先后顺序,ivs要在前面,因为是替换openPdf里面的类
```
<dependency>
    <groupId>com.github.doobo</groupId>
    <artifactId>openpdf-ivs</artifactId>
    <version>1.0</version>
</dependency>
 
<dependency>
    <groupId>com.github.librepdf</groupId>
    <artifactId>openpdf</artifactId>
    <version>1.3.20</version>
</dependency>
```
